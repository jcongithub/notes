Spring Data JPA and Hibernate
- JPA: Java Persistence API
- Hibernate Framework: object-relational mapping (ORM) is the programming technique to map application domain model objects to the relational database tables.. Hibernate is an implementation of JPA
- Spring Data JPA: not a JPA provider. it is a library/framework that adds extra layer of abstraction on the top of JPA provider. Spring Data JPA always requires the JPA provider such as Hibernate
